# Triage Labels

Matt Pocock workflow skills use five canonical triage roles. This file maps
those roles to the labels used in this repository's GitHub Issues.

| mattpocock/skills role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | A maintainer needs to evaluate the issue. |
| `needs-info` | `needs-info` | Waiting for the reporter to provide more information. |
| `ready-for-agent` | `ready-for-agent` | Fully specified; Codex can take it without more human context. |
| `ready-for-human` | `ready-for-human` | Needs human implementation or product judgment. |
| `wontfix` | `wontfix` | Will not be implemented. |

When a skill mentions a role, use the GitHub label in the right column.
